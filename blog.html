Finishing ICTU Project

11/30/2020

by Jacob Tener

I've just concluded the process of wrapping up the Tenants Union project I've been working on for the past 6 weeks. The project is hosted on Heroku, and the client preferred to save as much money as possible, so we purchased a domain that simply redirects to the heroku domain. It was a lot of fun to learn more about how well JavaScript can work as a Backend language with Node.js. I also enjoyed learning about routing within MVC architecture, and how to implement passport security with sessions to manipulate routing. The site can be found here or on my main page.
